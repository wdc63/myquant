# MyQuant 平台用户指南 (V2 - 完善版)

## 1. 引言

欢迎使用 MyQuant 在line量化回测平台！

本项目旨在为您提供一个私有化、轻量级、功能完备的量化策略开发与回测环境。平台深度集成了 `qtrader` 核心回测引擎，并通过可插拔的 DataProvider（数据合约）接入你自己的数据源/实盘接口（开源版不内置私有 SDK）。您无需关心繁琐的环境配置和数据对接，即可专注于策略的研发与验证。

本指南将详细介绍平台的**环境准备**、**安装**、**配置**、**启动**和各项功能的**使用方法**。

---

## 2. 环境准备

在开始安装之前，请确保您的开发环境满足以下要求。

### 2.1 Python 环境

*   **版本要求**: Python `3.9` 或更高版本。
*   **获取方式**: 如果您尚未安装 Python，请从 [Python 官方网站](https://www.python.org/downloads/) 下载并安装。
*   **验证安装**: 打开终端并运行 `python --version` 和 `pip --version`，确保命令能正常执行。

### 2.2 Node.js 环境

*   **版本要求**: Node.js `20.19.4` 或更高版本 (同时会自动安装 `npm`)。
*   **获取方式**: 请从 [Node.js 官方网站](https://nodejs.org/en/download/) 下载并安装 **LTS (长期支持)** 版本。
*   **验证安装**: 安装完成后，打开一个新的终端并运行 `node -v` 和 `npm -v`，确保命令能正常执行。

---

## 3. 安装步骤

环境准备就绪后，请按照以下步骤安装平台。

### 3.1 后端安装

后端服务是整个平台的核心。

#### 3.1.1 安装第三方依赖

所有必需的 Python 第三方库都已在 `myquant/requirements.txt` 中列出。在**项目根目录**下，打开终端并执行以下命令：

```bash
pip install -r myquant/requirements.txt
```

#### 3.1.2 安装核心本地 SDK

本平台依赖于三个核心的本地 Python 包。您需要将它们以“可编辑模式” (`-e`) 安装到您的 Python 环境中，这可以确保平台能正确地引用它们。

在**项目根目录**下，依次执行以下三条命令：

```bash
pip install -e ./qtrader
```

### 3.2 前端安装

前端提供了所有可视化交互界面。

1.  **进入前端项目目录**:
    ```bash
    cd myquant/frontend
    ```

2.  **安装 Node.js 依赖**:
    此命令会下载并安装所有前端所需的库，如 Vue.js, Element Plus, Monaco Editor 等。
    ```bash
    npm install
    ```

## 4. 平台配置 (重要)

平台的所有关键配置都集中在 `myquant/myquant_config.json` 文件中。**在首次启动前，请务必根据您的实际情况修改此文件。**

```json
{
  "auth": {
    "password": "admin123" 
  },
  "server": {
    "host": "127.0.0.1",
    "port": 5000
  }
}
```

### **初始密码说明**

*   **位置**: `myquant/myquant_config.json` 文件中的 `auth.password` 字段。
*   **默认值**: `admin123`
*   **操作**: **请在首次启动前，务必将此默认密码修改为您自己的强密码。**

---

## 5. 启动平台

平台由独立的后端和前端服务组成，您需要分别启动它们。建议您打开两个终端窗口来执行此操作。

### 5.1 启动后端服务

在**项目根目录**下，运行以下命令：
```bash
python myquant/backend/app.py
```
当您看到 `启动 MyQuant Platform 服务器...` 的日志输出时，表示后端已成功启动。

### 5.2 启动前端服务

在另一个终端窗口中，进入 `myquant/frontend` 目录，然后运行：
```bash
npm run dev
```
此命令会启动一个前端开发服务器。终端会显示一个本地访问地址 (通常是 `http://localhost:5173`)。

现在，您可以通过浏览器访问前端服务的地址来使用 MyQuant 平台了。
